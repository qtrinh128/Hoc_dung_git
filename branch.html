Làm việc với Branch

Hiển thị danh sách branch: git branch

Tạo 1 branch: git branch ten_branch

Để chuyển đổi qua lại giữa các branch: git checkout ten_branch

Lưu ý: Việc dev trên 1 nhanh mới sẽ không ảnh hưởng gì tới những nhánh khác

Gộp dữ liệu từ 1 branch

Ví dụ cần chuyển dữ liệu từ branch A sang branch master. Điều kiện cần là dữ liệu trên nhánh A phải được commit(không cần push)

Sau đó ta tiến hành gộp dữ liệu như sau

git checkout master

git merge A

Như vậy là tiến hành gộp dữ liệu từ nhánh A vào nhánh master thành công., sau đó ta có thể push lên

Xóa Branch

Lưu ý: branch cần xóa phải được gộp dữ liệu về master

Ta sử dụng lệnh: git branch -d ten_nhanh
