Đánh dấu commit với tag

Nếu có quá nhiều commit sẽ gây khó khăn nếu cần xem lại lần commit trước vì gắn tag cho commit sẽ rất hữu ích để tra cứu thông tin 1 các rõ dàng

Sử dụng lệnh: git show ten_tag

Có hai loại tag

  • Lightweight tag: Các tag này chỉ đơn thuần là đánh dấu các snapshot của mỗi commit
  • Annotated tag: Có thể đặt tiêu đề, và khi xem sẽ có thông tin về người tag và ngày tag

Tạo lightweight tag

Trước tiên gõ: git tag để xem danh sách các tag có trong dự án

Để tạo 1 tag gõ lệnh: git tag ten_tag

Lưu ý rằng sau khi gõ git tag ten_tag || Ví dụ: git tag v1 thì lần commit cuối cùng tức lần commit trước khi tạo tag v1 sẽ được gắn vào v1

Tạo Annotated tag

Gõ: git tag -a v1 -m "message" để tạo tag v1 với lời nhắn message

Chú ý: mặc định khi push sẽ KHÔNG push các tag lên repository mà ta phải dùng lệnh: git push --tags để đẩy toàn bộ tag lên
