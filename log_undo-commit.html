Log và Undo commit

Log

Để xem lịch sử commit ta sử dụng lệnh: git log

Một vài option

  • Hiện thị chi tiết mỗi lần commit git log -p
  • Để xem 1 lần commit gần nhất git log -1
  • Xem commit của một người nào đó git log --autho ten--nguoi-commit-nmuon-xem

Lọc log với pretty

Sử dụng pretty rất hay nếu bạn muốn lọc chỉ những cái cần xem khi view log

Với câu lệnh: git log --pretty "tag_name"

Danh sách các tag

  • %H: Chỉ hiện commit hash
  • %h: rút ngắn của hash
  • %an: Tên người commit
  • %ae: Email commit
  • %s: Nội dung commit
  • ...
