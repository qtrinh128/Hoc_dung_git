Chú ý khi làm việc với branch

Giả sử ta đang làm việc trên một nhánh nhanh1 của project không phải master

Để lấy toàn bộ code trên nhánh master về nhanh1 của ta thì ta cần check xem ta đang ở nhanh nào bằng lệnh: git status

Sau đó nếu đúng đang làm việc trên nhánh nhanh1 của ta mà muốn có cả code ở phần master thì trong nhanh1 ta sử dụng lệnh: git pull origin master
