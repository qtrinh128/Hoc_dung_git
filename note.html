Chú ý

Nội dung chỉnh sửa hiện tại của file là: Lần chỉnh sửa 1 ABC

Giả sử file này đã được add bằng lệnh git add *

Tuy nhiên ta chưa commit

Sau đó ta tiếp tục chỉnh sửa file này và thêm bớt nội dung của nó

Sau đó nếu ta không tiến hành thực thi lênh: git add * 1 lần nữa mà commit luôn

git commit -m 'message' thì nội dung: Lần chỉnh sửa 1 ABC sẽ được commit chứ không phải nội dung hiện tại ở lần chỉnh sửa tiếp theo
