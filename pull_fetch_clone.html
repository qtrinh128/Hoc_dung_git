Sự khác nhau giữa pull fetch clone

git clone Lệnh này sẽ sao chép toàn bộ dữ liệu trên repository và các thiết lập về repository tức là nó sẽ tự động tạo 1 branch trên máy tính cá nhân của ta

Lệnh này chỉ nên dùng khi cần tạo 1 Git mới trên máy tính với toàn bộ dữ liệu và thiết lập của một remote repository

git pull Lệnh này sẽ lấy toàn bộ dữ liệu từ remote repository và gộp vào branch hiện tại đang làm việc

git fetch Lệnh này sẽ lấy toàn bộ dữ liệu từ remote repository nhưng cho phép ta gộp thủ công vào 1 branch nào đó trong máy tính của mình
